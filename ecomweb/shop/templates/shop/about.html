My Awesome Cart

In this project I have used Django web framework of python to build a e-commerce website which functions almost like the other e-commerce websites out there in the market. This being one of my major projects has high complexity with respect to the code although the code is understandable.This website can do various operations like you can add the products to cart, view each product seperately and also review them by giving a star rating and writing down your review, then you can go to the cart to increase the quantity of the goods in your cart or remove them from your cart also you will be shown the grand total. Then finally you can checkout but for that also you need to be logged in and once you are logged in you will be shown the checkout page wherein you can fill the details although I have not integrated payment gateway because this is just a trial website. After you have placed you order you will get the order id which you can use to track the order.
